Entries test page

This page tests how your browser handles entries, keys and values defined in ES2015 and ES2017. Object has several functions form accessing the properties of an object. The keys function was introduce in ES2015, it returns an array of the property names. The values function was introduce in ES2017, it returns an array of the property values. The entries function was introduce in ES2017, it returns an array of the pairs of property keys and values.

Example

const obj = { prop: 'hello' }; Object.keys(obj); // returns an array containing the string 'prop' Object.values(obj); // returns an array containing the string 'hello' Object.entries(obj); // returns an array containing the array ['prop', 'hello']
