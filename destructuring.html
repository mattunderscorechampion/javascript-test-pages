Destructuring

Destructuring is a away take apart complex objects. Arrays and objects can be destructured to access the values they contain.

Examples

// Assign 'x' to a and 'y' to b var [a, b] = ['x', 'y']; // Assign 'x' to c and 'y' to d var [c ,, d] = ['x', 5, 'y']; // Assign 'x' to e and 'y' to f var [e, f = 'y'] = ['x']; // Assign 'x' to a and 'y' to b var {a, b} = {a: 'x', b: 'y'}; // Assign 'x' to c and 'y' to d var {c: t, d: v} = {c: 'x', d: 'y'}; // Assign 'x' to e and 'y' to f var {e, f = 'y'} = {e: 'x'}; // Assign 'x' to g and '{h: "y", i: "z"}' to h var {g, ...h} = {g: 'x', h: 'y', i: 'z'}
