padStart and padEnd test page

This page tests how your browser handles padStart and padEnd defined in ES2017. Strings have introduced padding functions. These functions extend strings with additional characters at the start or end. You may have heard of similar functionality previously being available in an NPM package.

Example

'hello'.padStart(10); // Prefix the string with 4 spaces 'hello'.padEnd(10, 'xy'); // Suffix the string with 'xyxy'
