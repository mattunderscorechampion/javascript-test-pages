Var, let and const

This page tests how your browser handles different ways JavaScript can declare symbols. The var keyword has always been supported, it defines a function scoped, mutable symbol. The let keyword was defined in ES2015, it defines a block scoped, mutable symbol. The const keyword was defined in ES2015, it defines a block scoped, immutable symbol. Only symbols defined by the var keyword can be redefined.

Example

var x = 'The let keyword defines defines a function scoped symbol'; let y = 'The let keyword defines defines a block scoped symbol'; const z = 'The const keyword defines defines a block scoped symbol that cannot be reassigned or redefined';
