Arithmetic test page

This page tests how your browser handles arithmetic. Arithmetic, you know, working with numbers. JavaScript represents all numbers as double precession floating point. This can cause incorrect results.

Example

const addition = 2 + 3; const subtraction = 2 - 3; const multiplication = 2 * 3; const division = 4 / 2; const exponent0 = Math.pow(4, 2); const exponent1 = 4 ** 2;

Results

This table shows the result of arithmetic expressions
Expression Result
2 + 3 Failed to parse script
2 - 3 Failed to parse script
2 * 3 Failed to parse script
4 / 2 Failed to parse script
Math.pow(4, 2) Failed to parse script
4 ** 2 Failed to parse script
4 / 0 Failed to parse script
