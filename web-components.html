Web components

Web components provide a way to create reusable widgets that can be used on webpages. Browsers provide several distinct features to support them. Custom elements allows a component to declare its own HTML elements, these can be used in markup to include the component. Shadow DOM encapsulates DOM and CSS, it allows the components to be described in HTML separate from the rest of the document. HTML templates provide a way to include an HTML fragment that is not rendered in the document, it can be used to define the component. HTML imports provide a way to package web components.

There is a lot to know about web components. Rather than explaining all about them here you should check out some other sources.
